# User manual generator

Generates `docs/IWBF_Team_Points_Control_User_Manual.docx` — an English,
fully illustrated user manual for the app.

The screenshots are **faithful tablet-resolution reproductions** of the real
Flutter UI. They are built from HTML/CSS that mirrors each widget's exact
colours (`IwbfColors`), sizes, texts, Material icons, the real IWBF logo /
court assets and Unicode flag emoji, populated with the data from the
official sample spreadsheet generated by `TemplateGeneratorService`
("IWBF America's Cup", Brazil - Men vs Argentina - Men).

## Files

- `app.html` — every screen/state, rendered at 800×1280 dp (tablet portrait),
  plus the Excel mock-ups and the phone (tabs) layout.
- `render.mjs` — drives headless Chromium to screenshot each screen into
  `screenshots/`.
- `build_docx.py` — assembles the Word document (text + screenshots + tables).
- `screenshots/` — the generated PNGs (committed).

## Regenerating

Prerequisites (the manual was built in a Linux sandbox):

```bash
# 1. Fonts (Roboto = Flutter's font; Material Icons = Flutter's glyphs)
mkdir -p tools/manual/assets/fonts
npm install @fontsource/roboto@5 @fontsource/material-icons@5
cp node_modules/@fontsource/roboto/files/roboto-latin-{400,500,700,900}-normal.woff2 tools/manual/assets/fonts/
cp node_modules/@fontsource/material-icons/files/material-icons-latin-400-normal.woff2 tools/manual/assets/fonts/material-icons.woff2
# Flag emoji come from the system Noto Color Emoji font (apt: fonts-noto-color-emoji)

# 2. App assets (logo + court). Court is pre-rotated 90° to match the app's RotatedBox.
cp assets/images/iwbf-logo-black.png assets/images/iwbf-logo-horizontal-black.png assets/images/court.png tools/manual/assets/
python3 -c "from PIL import Image; Image.open('tools/manual/assets/court.png').rotate(-90, expand=True).save('tools/manual/assets/court_portrait.png')"

# 3. Render screenshots (needs Chromium via playwright)
node tools/manual/render.mjs

# 4. Build the .docx (needs: pip install python-docx Pillow)
python3 tools/manual/build_docx.py
```
